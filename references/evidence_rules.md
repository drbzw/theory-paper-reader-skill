# Evidence and Source-Location Rules

## 1. Three epistemic statuses

Use these labels consistently:

- **PAPER-EXPLICIT** — the source directly states the claim.
- **MODEL-INFERRED** — the claim follows from the paper's formal structure but is not explicitly named.
- **CANONICAL-BACKGROUND** — the claim concerns the broader model family rather than this paper.

Use `NOT SPECIFIED` when the paper does not define an item. Do not fill the gap from memory.

## 2. PDF locators

For prepared PDF text, cite normalized page-line locations such as `P012-L004–P012-L026`. Add the printed page number, section, equation, proposition, theorem, lemma, or appendix label when available.

A page number alone is acceptable for broad discussion, but load-bearing assumptions, equations, propositions, and timing claims should use a narrower locator.

## 3. Markdown locators

For Markdown, including MinerU output, use the nearest heading plus normalized line locations such as:

`Section “2. Model,” MD-L00482–MD-L00531`.

If the Markdown retains page markers, record them as supplementary evidence. Do not invent page mappings when none exist.

Relative image links, equation images, and tables may contain essential model information. Inspect them when available and relevant. If an asset is missing or unreadable, state the limitation.

## 4. When both PDF and Markdown are available

Use Markdown as the primary text if it is cleaner. Use PDF to verify printed page locations, figures, diagrams, and equation layout. Record material discrepancies instead of silently reconciling them.

## 5. Claims requiring direct evidence

Direct source evidence is required for:

- agent roles and objectives;
- sequence of moves;
- private and public information;
- commitment assumptions;
- payoff functions and constraints;
- equilibrium definitions;
- assumptions used in propositions;
- proposition and welfare claims;
- statements about what the paper inherits or changes;
- claims that the authors follow or extend a specific ancestor.

## 6. Quotes and paraphrases

Prefer concise paraphrases. Use short quotations only when wording is necessary to establish lineage or a precise assumption. Never use a quotation as a substitute for explaining the economics.

## 7. Conflicts and uncertainty

When the introduction's verbal description conflicts with the formal setup, prioritize the formal model and report the tension. When proof assumptions differ from the proposition statement, flag the narrower proof conditions. When evidence is incomplete, reduce confidence rather than completing the model from textbook expectations.
