# Seed Map of Relevant Model Families

This is a search aid, not a closed taxonomy. Always permit families absent from this list.

- signaling;
- hidden-action moral hazard;
- hidden-information moral hazard;
- career concerns and reputation;
- multitask agency;
- team production and free riding;
- common agency;
- dynamic principal-agent models;
- mechanism design;
- relational contracts;
- renegotiation and commitment;
- incomplete contracts and property rights;
- authority, delegation, and real authority.

## 6. Classification discipline

Use the narrowest defensible subfamily only after identifying the broader skeleton. A paper can have one primary skeleton and several secondary mechanisms. The empirical setting—banks, auditors, local officials, AI disclosure, venture capital—is an application domain unless it changes the formal structure.
